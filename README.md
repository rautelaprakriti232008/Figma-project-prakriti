# 🚀 Programming Pathshala -- Mock Assessments Landing Page

A fully responsive and modern landing page built using **HTML5** and
**CSS3**.\
This project replicates a real-world EdTech platform UI with structured
sections, reusable components, and responsive layouts.

------------------------------------------------------------------------

## 📌 Project Overview

This landing page includes:

-   Responsive Navigation Bar
-   Hero Section
-   Why Choose Us Section
-   Mock Assessments Grid (3x4 Layout)
-   Topic Based Assessments
-   Courses Section
-   Advanced Responsive Footer Layout
-   Mobile-Friendly Design

The layout is built using **CSS Grid**, **Flexbox**, and **Media
Queries** to ensure responsiveness across all screen sizes.

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   HTML5
-   CSS3
-   Flexbox
-   CSS Grid
-   Media Queries
-   Google Fonts (Poppins)
